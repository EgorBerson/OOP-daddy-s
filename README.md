* Вместо метода по определению среднеарифметического применил функцию average()  для всех расчетов среднего. Метод мог бы выглядеть примерно так:

  ```
    def __average(self):
        sm = 0
        ln = 0
        for values in self.grades.values():
            sm += sum(values)
            ln += len(values)
        if ln == 0:
            return 'No grades'
        return round((sm / ln),1)
